# UNP_PROJECT_BI_WITH_POWER_BI_SUPPLY_CHAIN-_DATA_ANALYSIS

This project focuses on Supply Chain Analysis using Microsoft Power BI to transform raw data into meaningful business insights. The dashboard evaluates sales performance, product revenue, and regional distribution to understand market trends. It also analyzes inventory levels and supplier lead times to improve operational efficiency. Cost components such as manufacturing and logistics are examined to identify expense patterns. Product quality is assessed using defect rate analysis to ensure performance standards. Interactive visualizations and KPIs help monitor key supply chain activities effectively. Overall, the project supports data-driven decision-making and enhances supply chain efficiency.
